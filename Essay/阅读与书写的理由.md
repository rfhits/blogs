# 阅读的理由

收藏的文章已经读不完了，为了能够更好地理解这些文章，我需要想清楚自己为什么阅读。

1. 思考问题。有些文章让我对一些问题产生思考。
2. 看到另一种人（生）。书写的人，有人是记者，有人是科研工作者，或许是我的生活太无聊、性格太闷，我渴望了解其他人的性格和眼中的世界。
3. 接触新观点。相左的观点可以防止过偏，全新的观点可以开阔眼界。
4. 找到共鸣。作者可以准确地表达出我内心的想法，我写不出来。发觉世界上（曾经）有个人和自己「心有灵犀」的感觉很神奇，真的有人会和我的想法、行为、习惯一致。
5. 提供指导。文章有时会直接指导我「应该做什么」与「不应该做什么」。
6. 文字的魅力。我只能用苍白的语言描述那些文字对我的感受：「我觉得写得很好」、「我这辈子都写不出来这种东西」……
7. 更轻的「负罪感」。刷视频、打游戏等娱乐活动，属于[「easy and fun」](https://www.youtube.com/watch?v=arj7oStGLkU&t=638s)的范畴，事后回想起来会有很重的负罪感。
8. 缓解焦虑。阅读的时候在思考，能够短暂地忘记一些烦恼，作者幽默的语言也能带来快乐。

这些理由或给我带来阅读上的快乐，或影响我未来的决策。

|        理由        | 阅读时的快乐程度 | 未来的决策影响 |
| :----------------: | :--------------: | :------------: |
|      思考问题      |        0        |       5       |
| 看到另一种人（生） |        1        |       3       |
|     接触新观点     |        1        |       3       |
|      找到共鸣      |        4        |       0       |
|      提供指导      |        1        |       4       |
|     文字的魅力     |        5        |       1       |
|  更轻的「负罪感」  |        1        |       0       |
|      缓解焦虑      |        1        |       0       |

阅读促使我思考，但是思考大多数时候时严肃的，很难给我带来快乐，却给我的未来带来最大的影响；感受文章的文字魅力、和作者共情，能让我得到不少的快乐，但几乎不会对我未来的决策产生影响。将这些理由综合起来考虑，有些文章能够在多个维度上平衡，既能让我接触到新的观点，也能给我共鸣，让我体会到文字的魅力。

# 书写的理由

缘起是被问起为什么写日记，在看完[「我为什么写博客」](https://tianxianzi.me/2022/12/29/why_do_i_write_blogs/)后，我开始认真思考书写的理由。

按照我的拙见，写作内容可以分为记录和抒发，我的大多数文字都只是在记录，很少表达自己的情感和观点。

按重要性排序，我书写的理由如下：

1. 放松自己、缓解焦虑。写作可以暂时逃避现实的压力，看着墨水从笔尖流出，汉字留在本子上，有一种安心的感觉。我觉得汉字非常奇妙，我的祖先和我在用类似的符号表达类似的感情，我甚至觉得短信、邮件胜过微信和 QQ 消息，它更贴近古人的问候方式，生日祝福、教师节祝福，我分别使用邮件和短信。
2. 记录现在，对抗遗忘。我讨厌想不起来某件事甚至记错某件事的感觉，就好像被人打了却不记得那个人是谁。
3. 奢望自己回看的时候有什么新的体会。

我好像不曾为启发读者的思考而写过什么，也没有给读者提供什么新颖的观点，更不必提「文字的魅力」。在阅读理由中排名最低的「缓解焦虑」，竟然在书写的理由中排第一。
