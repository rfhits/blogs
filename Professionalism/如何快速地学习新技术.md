如何快速地学习新技术

笔记于韩顺平的教学视频：[如何学习 Golang 更高效](https://www.bilibili.com/video/BV1ME411Y71o/?p=3)

分为 5 个步骤：

1. 新技术源于需求
2. 解决需求
3. 了解新技术
4. 快速入门案例
5. 深入学习技术

# 新技术源于需求

项目开发时，面临一个问题，需要解决。

如 n 年前的“页面局部刷新问题”，用户不需要整个门户的页面，只会浏览部分。

# 解决需求

当使用传统的方法解决时，不够好，所以会考虑使用新技术。

传统的技术 iframe 能够解决“页面局部刷新问题”，但是不如 Ajax。

# 了解新技术

了解技术的基本原理和语法，不涉及技术细节。

如“Ajax 结合了 dom 和 js”

# 快速入门案例

不需要涉及细节

# 深入技术或知识点的细节

1. 使用规范
2. 使用陷阱
3. 使用时，要注意什么
