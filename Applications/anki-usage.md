# anki 批量选择题

参考 [anki\|批量制作选择题的另一种思路 \- 知乎](https://zhuanlan.zhihu.com/p/47965443)

1. 找到并使用自己喜欢的模板。只需要导入一份卡组就会存储相应的模板，要确定自己的模板使用的表格格式是怎样的。
   我选择的模板是 [anki 选择题模板—自动匹配单选多选—Monokai 风格—带计分板 \- 知乎](https://zhuanlan.zhihu.com/p/104245490)，对应的 csv 格式一定要有 remark 列。
2. 一般内容的来源是纯文本 word，关键是从 word 内容转换为表格的形式
   参考 [Excel 每 N 列数据转换为一行显示，这个方法值得收藏！\_Excel 情报局的技术博客\_51CTO 博客](https://blog.51cto.com/u_15825298/5757995)
3. 使用方方格子这个 Excel 插件，以换行符为分隔符合并不同选项所在单元格
4. 复制粘贴 excel 内容到 vscode 里，会自动转换为 UTF-8 的格式。Microsoft Excel 的 csv 导入肯定存在 bug，vscode 更快
5. anki 导入文件，选择模板和文件

总的来说，就是要实现一个工作流，可以处理以下输入和输出：

输入普通文本的 word：

```
【单选题】 张静老师在《漫谈青年知识分子的成长》中谈到，历史学的最终的目的、最高意图是_______。
A、满足个人求知欲
B、生产历史知识
C、推动学科进步
D、探求历史的真实
D

【单选题】 张静老师在《漫谈青年知识分子的成长》中谈到，治史的最好顺序要先从_______入门。
A、通史
B、专史
C、世界史
D、国内史
A

【单选题】 梁启超认为，要想做一个历史学家，必须具备四种资格，就是_______，这样的一个顺序。
A、才、学、识、德
B、才、识、学、德
C、德、学、识、才
D、德、才、学、识
C
```

输出带换行符的 csv：

```csv

C、修身齐家
D、治国平天下"				B	empty-remark

B、才、识、学、德
```
